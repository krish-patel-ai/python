## 🧠 Philosophy

Most people learn AI by watching videos and running notebooks.
I learn by **breaking things and building them back.**

Rules I follow in this repo:
- Every algorithm gets implemented from scratch **before** using a library
- No topic is marked done until I can build something with it
- Commit messages are meaningful — not just "update"
